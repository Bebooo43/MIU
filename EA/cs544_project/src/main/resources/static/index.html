Testing Endpoints

Because of Spring Security, you need to put the Basic HTTP Authentication the respective uses, accordingly with the service

  •     ### update session
    PATCH https://cs544-project.herokuapp.com/admin/sessions/1
    Authorization: Basic admin 1234
    Content-Type: application/json
    
    {
      "location": "Library"
    }
    
  •         
    ### get session (belonging to provider) by id
    GET https://cs544-project.herokuapp.com/provider/sessions/3
    Authorization: Basic provider 1234
    Accept: application/json

      
  •     ### approve appointment
    PUT https://cs544-project.herokuapp.com/provider/appointments/1
    Authorization: Basic provider 1234
      
  •     ### get available (future and no approved)
    GEThttps://cs544-project.herokuapp.com/provider/sessions/available
    Authorization: Basic provider 1234
    Accept: application/json
      
  •         
      ### provider creating session
      POST https://cs544-project.herokuapp.com/provider/sessions

      Content-Type: application/json

              "startTime": "11:00:00",
              "duration": 120,
              "location": "Verill Hall"
      }
